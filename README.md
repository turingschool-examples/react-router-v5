# React Router v5 Lesson Repo

This repo contains exercises for the React Router v5 lesson.

## Installation Instructions
```bash
# Clone the repo
cd react-router-v5
npm i
npm start

# Navigate to localhost:3000 in your browser
# open your text editor
```
